**[Role]**
너는 예술가 브랜딩 전문 프론트엔드 개발자이자 UI/UX 디자이너야. 
이현정 대표의 전문성과 예술적 감각이 느껴지는 '인터랙티브 경력(Experience) 섹션'을 제작해줘.

**[Tech Stack]**
- React, Tailwind CSS, Framer Motion (필수)
- Icons: Lucide-react

**[UI/UX 상세 요구사항]**
1. **레이아웃 구조:**
   - 웹(Desktop): 좌측 카테고리 탭 메뉴 / 우측 이력 카드 (Horizontal Layout).
   - 모바일(Mobile): 상단 가로 스크롤 카테고리 칩 / 하단 이력 카드 (Vertical Layout).
2. **연도 표기 (중요):**
   - 모든 항목은 **[활동 내용 | 연도]** 순서로 배치할 것.
   - 연도(예: 2013 ~ 2023)는 텍스트 뒤쪽에 위치하며, `text-secondary/50` 컬러와 약간 더 작은 폰트 사이즈를 적용해 시각적 계층 구조를 형성할 것.
3. **인터랙티브 배경색 트랜지션:**
   - 카테고리를 변경할 때마다 섹션 전체의 배경색이 아주 은은하게 변화해야 함 (예: 합창-Slate50, 뮤지컬-Blue50, 안무-Stone50 등). 
   - `transition-colors duration-700`을 사용하여 부드럽게 전환되도록 할 것.
4. **예술적 배경 그래픽:**
   - 각 카드 내부 배경에 매우 연한(`opacity-[0.03]`) 음악 악보나 지휘봉 선 형태의 SVG 패턴을 삽입하여 예술적 아이덴티티를 부여할 것.
5. **애니메이션 효과:**
   - 탭 전환 시 `AnimatePresence`와 Framer Motion의 `staggerChildren`을 활용하여 리스트 항목이 아래에서 위로 순차적으로 올라오는(Fade-in up) 효과 적용.
